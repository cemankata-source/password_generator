# password_generator
python projek for password generator


Generate secure passwords with customizable length and symbol options. Includes a password strength checker.

## Features
- Custom password length
- Optional symbols
- Strength evaluation (Weak / Medium / Strong)
- CLI interface (terminal based)

## How to Run
1. Install Python 3
2. Open terminal
3. Run script:
   ```bash
   python password_generator.py
